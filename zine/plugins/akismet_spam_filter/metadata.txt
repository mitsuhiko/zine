Description: <a href="http://akismet.com/">Akismet</a> checks your comments\
 against the Akismet web service to see if they look like spam or not. It\
 also allows you to contribute back by reporting spam/ham. You need a\
 <a href="http://wordpress.com/api-keys/">WordPress.com API key</a>\ to use it.\
 You can review the spam it catches under "Comments" after configuring it.
